# East Port City Distribution - Dashboard

This project contains a custom KPI dashboard for East Port City Distribution, showing comprehensive analytics for their THCA & Tobacco wholesale business.

## Features

- Real-time KPI tracking across all business functions
- Inventory alerts and sales spike detection
- Custom growth projections & scenario modeling
- Actionable insights for business growth
- Dark mode interface for reduced eye strain

## Getting Started

### Prerequisites

- Node.js 14.0.0 or later
- npm 6.14.0 or later

### Installation

1. Clone the repository
   ```
   git clone https://github.com/yourusername/eastport-dashboard.git
   ```

2. Install NPM packages
   ```
   cd eastport-dashboard
   npm install
   ```

3. Start the development server
   ```
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

## Deployment

This project is ready to be deployed with Vercel or Netlify:

1. Push to GitHub
2. Connect to Vercel/Netlify
3. Deploy
